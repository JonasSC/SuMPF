Detailed Description

A class for saving a Signal to a file.

If NumPy is available, this class can write Signals in NumPy's npz-format. If PySoundFile or scikits.audiolab is available, this class supports wav, aiff, flac and ogg as well. If oct2py is available, it it possible to write Signals to Matlab .mat files. The structure of these files is described in the documentation of the class sumpf.modules.SaveSignal.MATLAB
