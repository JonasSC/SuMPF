Detailed Description

A class for saving a Spectrum to a file.

If NumPy is available, this class can write Signals in NumPy's npz-format. If oct2py is available, it it possible to write Signals to Matlab .mat files. The structure of these files is described in the documentation of the class sumpf.modules.SaveSignal.MATLAB
