Detailed Description

Objects of this class generate spectrums whose magnitude is falling with a given slope.

The slope is defined by the function 1/(f**x), where f is the frequency and x is an exponent that can be chosen freely. It is also possible to define a start frequency, before which the magnitude is constantly one. The phase of the output spectrum is always zero.
