Detailed Description

A simple progress dialog.

Other than the wx.ProgressDialog, this one does not offer a "Cancel"-button and it has a different way of setting the progress. This dialog is meant to be connected to a progress indicator, to track the progress of a calculation of a progressing chain.

def _sumpf._gui.progressdialog.ProgressDialog.SetProgress (   self,
  progress 
)

This method can be connected to a progress indicator's GetProgressAsTuple method, so this Dialog can keep updated about the progress of a processing chain's calculation.

The parameter progress can be either a tuple with two values (max, current), where max is the total number of tasks, that have to be completed and current is the number of those, which have finished. If progress is a three-element tuple (max, current, message), the message of the progress dialog will be updated to the value of the third parameter.
