Detailed Description

A class for generating a Signal from an arbitrary time function.

The given function must accept a time value as a parameter and return the sample of the Signal's channel at that point in time. Furthermore, a label can be specified for the generated Signal's channel. There can be only one function specified per instance of this class. Consequently, the generated Signal will have only one channel.
