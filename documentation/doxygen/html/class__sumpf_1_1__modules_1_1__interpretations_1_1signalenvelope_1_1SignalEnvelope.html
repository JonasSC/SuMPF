Detailed Description

A module that calculates the envelope of a given Signal.

Two modes are available:

  • UPPER calculates the envelope by half wave rectification and using the positive half wave. This results in the envelope above the given input Signal.
  • LOWER calculates the envelope by half wave rectification and using the negative half wave. This results in the envelope below the given input Signal.
