Detailed Description

Copies the channels of an input Signal to create an output Signal with a given number of channels.

The number of channels can either be specified as an integer number or by passing a ChannelData instance (e.g. Signal or Spectrum), whose channel count will be used.

If the Signal has multiple channels, all these channels will be copied in the correct order. This order will be repeated if necessary until the desired channel count is reached. This means that an output Signal with less channels than the input Signal is possible. Although it is recommended to use a SplitSignal instance for this purpose.

For example: input channels = c1, c2, c3; desired number of output channels = 5 => output channels = c1, c2, c3, c1, c2 input channels = c1, c2, c3; desired number of output channels = 2 => output channels = c1, c2
