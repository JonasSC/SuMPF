Detailed Description

A class for finding the maximum value(s) of a real valued data set, that is of derived from a Spectrum.

The data set can be the spectrum's magnitude, its phase, continuous phase or group delay or it can be the real or imaginary part of the spectrum. It is possible to retrieve the maxima for each channel individually or to find the one global maximum.
