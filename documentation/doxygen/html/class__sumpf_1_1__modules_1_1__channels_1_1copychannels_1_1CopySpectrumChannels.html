Detailed Description

Copies the channels of an input Spectrum to create an output Spectrum with a given number of channels.

The number of channels can either be specified as an integer number or by passing a ChannelData instance (e.g. Signal or Spectrum), whose channel count will be used.

If the Spectrum has multiple channels, all these channels will be copied in the correct order. This order will be repeated if necessary until the desired channel count is reached. This means that an output Spectrum with less channels than the input Spectrum is possible. Although it is recommended to use a SplitSpectrum instance for this purpose.

For example: input channels = c1, c2, c3; desired number of output channels = 5 => output channels = c1, c2, c3, c1, c2 input channels = c1, c2, c3; desired number of output channels = 2 => output channels = c1, c2
