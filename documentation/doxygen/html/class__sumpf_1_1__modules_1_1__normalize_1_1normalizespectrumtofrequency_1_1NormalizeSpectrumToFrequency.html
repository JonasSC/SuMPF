Detailed Description

Normalizes the channels of a Spectrum instance so that its magnitude at a given frequency is 1.0.

The value of the input Spectrum at the given frequency will be calculated by linearly interpolating the Spectrum's magnitude. If the magnitude for the given frequency is 0.0, the channel will not be normalized to avoid divisions by zero.

Note that this works differently compared to the NormalizeSignal module. A normalized Spectrum which is then inversely fourier transformed does not necessarily result in a normalized Signal.
