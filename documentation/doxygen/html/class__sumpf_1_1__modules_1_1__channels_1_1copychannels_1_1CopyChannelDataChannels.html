Sets the number of channels of the output data set.

This can be done either by specifying the integer number of channels or by passing a ChannelData instance (e.g. Signal or Spectrum), whose channel count will be used. If the input data set has multiple channels, all the channels will be copied until the channel count is reached. An example: input channels := A, B, C channel count := 8 output channels := A, B, C, A, B, C, A, B
