Detailed Description

An abstract base class for other LineManager implementations.

Instances of derived classes of this take care of the display of one line in a plot. Derived classes can implement the method _PlotDownsampledData to add automatic downsampling functionality to the plots. Each instantiatable LineManager class must also inherit from a XDataManager implementation to be able to generate the x axis data for the plot.
