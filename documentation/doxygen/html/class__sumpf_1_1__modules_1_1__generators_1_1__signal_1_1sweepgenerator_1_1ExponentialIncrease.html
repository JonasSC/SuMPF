static

This static method is called before actually generating the sweep.

It precomputes and returns values, that are independent of the sample, so that their use increases the computation speed of the individual samples.

Parameters
f0the start frequency of the sweep in Hz as a float
f0the stop frequency of the sweep in Hz as a float
Tthe duration, that it takes the sweep to go from the start to the stop frequency, in seconds as a float
Return values
a tuple of precomputed values
