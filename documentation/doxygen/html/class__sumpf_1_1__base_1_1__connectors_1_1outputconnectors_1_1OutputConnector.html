Detailed Description

A Connector-class that replaces getter methods, so they can be used to connect different objects.

◆ NoticeAnnouncement()

def _sumpf._base._connectors.outputconnectors.OutputConnector.NoticeAnnouncement (   self,
  connector 
)

This method is called by the InputConnectors, which are observed by this output, when they are about to change.

It is used to estimate when the connected inputs shall be notified. So unnecessary computations in forked connection chains can be avoided.

Parameters
connectorthe observed input connector
