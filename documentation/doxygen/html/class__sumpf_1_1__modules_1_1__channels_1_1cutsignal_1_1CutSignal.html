Detailed Description

A module for cutting out a part of a Signal.

The part, that is cut out of the Signal is defined by a SampleInterval, but it is possible to define the interval by passing a tuple, a float or an integer without creating a SampleInterval instance. See the SampleInterval's documentation for details about how to define intervals.

Specifies the behavior, when the given interval is so narrow, that the output Signal would not have a valid length.

The behavior is set by passing one of the following flags: ERROR: raise a ValueError EMPTY: return an empty Signal, that has the same number of channels, sampling rate and labels as the input signal FULL: return the complete input Signal

Parameters
on_invalid_lengtheither CutSignal.ERROR, CutSignal.EMPTY or CutSignal.FULL
