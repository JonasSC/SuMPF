Overrides of this methods shall create plots and return them in a data structure like the following: plots[GROUP][COMPONENT] = matplotlib.Axes instance plots is a list, plots[GROUP] is a dictionary.

Parameters

the plots in a data structure like described above

def _sumpf._modules._plotting.plotlayouts.PlotLayout.FormatData (   data,
  labels,
  components 
)
static

Overrides of this static method shall sort the input data and labels into a data structure that is the same as the one for the plots: data[GROUP][COMPONENT] = list_of_sequences (one sequence per line) labels[GROUP][COMPONENT] = list_of_labels data and labels are lists, data[GROUP] and labels[GROUP] are dictionaries.

Parameters
dataa dict of input the data: input_data[component] = list_of_sequences (one sequence per channel)
labelsa list of labels, one for each channel. A label can be either a string or None
componentsa list of component names in the order in which they shall be displayed from top to bottom
Return values
a tuple of (data, labels), where data and labels are data structures as described above
