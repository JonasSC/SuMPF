Detailed Description

A module that converts a duration in seconds to a length in number of samples.

This way the length of a Signal can be set in a more intuitive way. Furthermore the duration of a Signal can be decoupled from its sampling rate.

Specifies, if the length shall be rounded to an even integer.

This is useful, when a Signal with the given length shall be transformed to the frequency domain. The fourier transformation expects the length of the Signal to be even. When transforming a Signal with an odd length, the last sample of the Signal will be dropped.

Parameters
even_lengthif True, the length is rounded to an even integer value, if False, the length is rounded to the integer that gives the closest match to the given duration
