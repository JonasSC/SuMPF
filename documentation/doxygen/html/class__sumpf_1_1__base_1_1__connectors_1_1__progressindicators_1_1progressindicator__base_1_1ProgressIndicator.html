Detailed Description

Base class for progress indicators.

These progress indicators can be used to track the progress of a calculation in a processing chain. It counts the number of involved methods and reports the progress each time one of these methods has finished running.

Derived classes have to implement the _Filter method that is used to select which methods shall be taken into account for tracking the progress. This is useful, if for example all setter methods just set some state variables, while the getter methods do the expensive calculations. In this case the setter methods should be filtered out and the progress should be calculated from the finished getter methods, to have a better estimate of the processing progress.
