Detailed Description

A module for calculating the integral of a Signal.

This is done by summing up the areas of the rectangles that are defined by the samples and the sampling rate. This method uses no interpolation between the samples.

Specifies an offset that shall be added to the output Signal's samples.

This offset can either be given as a float or it can be set to NO_DC in which case, the offset of each channel will be chosen so that the average of all the channel's samples is zero (this means that the channel has no dc-offset).

Parameters
offseta float or NO_DC
