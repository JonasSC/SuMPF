def _sumpf._base._connectors.basedecorators.TypedDecorator.GetType (   self)

This method returns the type of the Connector-method that is decorated with this decorator.

With this, the type of a Connector can be determined from a class and not only from an instance of that class. The decoration of methods still behaves a bit odd. When having a class, the determination of one of their method's type does not work the intuitive way: CLASS.METHOD.GetType() This will raise an error that None does not have the attribute METHOD. Instead, the following workaround has to be used: vars(CLASS)["METHOD"].GetType()

the type of the data that is passed through this connection
