def _sumpf._modules._channels.merge.MergeChannelData.SetLengthConflictStrategy (   self,
  strategy 
)

Sets the behavior for when the added data sets do not have the same length.

The following flags are available: RAISE_ERROR for raising a RuntimeError during merging, if data sets with different lengths have been added. FILL_WITH_ZEROS for solving the conflict by filling the too short data sets with zeros until all sets have the same length. CROP for cropping the length of all added data sets to the length of the shortest data set.

Parameters
strategyone of the merger class's flags for length conflict resolution
