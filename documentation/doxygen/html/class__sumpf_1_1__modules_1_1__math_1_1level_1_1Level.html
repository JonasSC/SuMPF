Detailed Description

A class for calculating the level of a signal.

The levels are computed individually for the channels and returned as a tuple. The computation of the level can be adjusted with the parameters power and root, which denote the exponent with which the powers of the input Signal's samples are computed before the integration and the degree of the root that is computed with the integrated values: level = (integrate(signal ** power)) ** (1/root) Here are some common examples for the coice of power and root: Computing the average amplitudes of the channels: power = 1.0, root=1.0 Computing the signal power of the channels: power = 2.0, root=1.0 Computing the RMS-level of the channels: power = 2.0, root=2.0
