Detailed Description

A class for Input connectors that can handle connections from multiple output connectors.

This could be the appropriate connector for add-methods. Requires an appropriate remove-method that reverts an add-method call. When the value of a connected output changes, this connector will call the remove-method to remove the old data, and then the add-method, which was replaced by this connector, to add the new data.
