Detailed Description

Takes two sequences of coefficients, that define a transfer function as a fraction in the laplace domain.

The resulting transfer function G(s) is computed as follows: a0, a1, a2, ..., an = numerator # the numerator sequence as it is passed to the constructor b0, b1, b2, ..., bn = denominator # the denominator sequence as it is passed to the constructor G(s) = (a0 + a1*s + a2*s**2 + ... + an*s**n) / (b0 + b1*s + b2*s**2 + ... + bn*s**n)
